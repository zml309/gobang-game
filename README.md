# 五子棋小游戏

一个机甲风格的五子棋小游戏，支持人机对战和AI难度调节。

## 功能特性

- 🎮 **游戏模式**：双人对战 / 人机对战
- 🤖 **AI难度**：简单 / 中等 / 困难
- 📱 **响应式设计**：适配PC和手机
- 🎨 **机甲风格**：蓝白科技感UI
- 💡 **棋子特效**：下落动画和发光效果
- 🔄 **游戏控制**：重新开始 / 悔棋

## 技术实现

- **前端技术**：HTML5 + CSS3 + JavaScript
- **AI算法**：Minimax算法 + Alpha-Beta剪枝
- **响应式布局**：媒体查询和弹性布局
- **视觉效果**：CSS3动画和渐变

## 如何玩

1. 打开游戏链接
2. 选择游戏模式（双人对战或人机对战）
3. 如果选择人机对战，选择AI难度
4. 开始游戏，点击棋盘落子
5. 黑棋先手，先连成五子的一方获胜

## 游戏规则

- 五子棋是一种两人对弈的纯策略型棋类游戏
- 黑白双方交替落子，先在一条直线（横、竖、斜）上形成连续的五颗棋子者获胜
- 落子位置必须是棋盘上的交叉点

## 开发说明

- 棋盘大小：15x15
- AI搜索深度：困难模式4层
- 响应式设计：适配手机和电脑屏幕
- 支持触摸操作：优化移动端体验
